# Personal Podcast

Daily news briefing — auto-generated by a scheduled Claude agent.

## Subscribe

**RSS feed:** `https://cristianomeda.github.io/personal_podcast/feed.xml`

Add this URL to any podcast app (Apple Podcasts, Overcast, Pocket Casts) to receive new episodes automatically. Works with CarPlay.

## How it works

A GitHub Actions workflow runs Monday–Friday at 7 AM CET:
1. Claude searches the web across configured topics
2. Ranks and synthesises the top stories into a digest
3. Sends an HTML email summary
4. Generates an MP3 via OpenAI TTS
5. Publishes the episode to this RSS feed
